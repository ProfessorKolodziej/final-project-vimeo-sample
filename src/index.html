Welcome to your final project!

This is a demo using the Vimeo player SDK. This library allows you to control a Vimeo video using JavaScript.

Check the console of this page using the inspector. You'll see the title of the video. When you play the video, a message will log to the console saying the video is playing.

All code is in js/scripts.js and is based off the Getting Started documentation. To get started with controlling the player, try using the Player Methods documentation.
